Angled and notched Cabs

These are some of the cabinets that we make once upon a little while and here are the basic spec's that we go by.

Goals:
  1. Add into Quickcam so we can have the machining done automatically, when these occur.
  2. Get proper times for appropriate pricing for differences like, when it’s off of the coveted 45 degree angle.
  3. Add all these options into QuickCAM to make processing easy for engineering.
  4. Add into Quoin to make processing easy for OE.
    1. AF’s (45 degrees)
    2. AS’s (45 degrees)
    3. AFC’s (45 degrees)

Standard custom angled cabinets:

AF (Angled Front Cabinet-45 degrees hinged on short gable)
  • AFLL (Angled Front Left side, Hinged Left)
  • AFRR (Angled Front Right side, Hinged Right)
  • The angle must be 45 degrees.
  • The front must be angled at 45 degrees (the width and difference between the depths must be equal).
  • Mounting plates = Blum 6mm (9mm high) –Blum code 175H9160.
  • Hinges =Blum 79B9458
  • Note that the mounting plate center is 37mm.
Process:
  • Preprogrammed for the Nesting machine.
  • Give to PA to pre-assemble.

Angled Front Cabinet-45 degrees hinged on long gable
  • AFLR (Angled Front Left side, Hinged Right)
  • AFRL (Angled Front Right side, Hinged Left)
  • The angle must be 45 degrees.
  • Mounting plates = 3mm (6mm high) 173H7130
  • Hinges =Blum 79B3450 (spendmap code = HINGE-45D)
  • Mounting Plate center = 55mm.
Process:
  • Preprogrammed for the Nesting machine.
  • Give to PA to pre-assemble.

Angled Front Cabinet-not 45 degrees
  • Cut gables ¼” larger.
  • Cut shelves exact size.
  • Dowel the short gable.
Process:
  • Preprogrammed for the Nesting machine.
  • Give to PA to preassemble.

AS (Angled Side-45 degrees)
  • The side must be angled at 45 degrees. (The width and the depth must be equal to each other).
  • Mounting plates = Blum 3mm (6mm high) (173H7130)
  • Hinges =Blum 79B3450 (spendmap code = HINGE-45D)
  • Note that the distance from the front of the cabinet to the center of the mounting plate system holes is 55mm.
Process:
  • Preprogrammed for the Nesting machine.
  • Give to PA to assemble.
