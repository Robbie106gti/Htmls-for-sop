Note that the difference between the thickness of the CC and the PL can be a lot in our kitchen cabinet world.

Composite core EP’s, EP-1VE-SQCC, EP-2VE-SQCC

Are usually very close, if not exactly 1 ½” / 38.1mm. This is before any veneer is applied.

Plywood core EP’s, EP-1VE-SQPC, EP-2VE-SQPC

Usually starts off thinner with a minimum thickness of, e.g 1 7/16” / 36mm and from sheet to sheet can be up 38.5mm in thickness. This is before any veneer is applied.

Usually this won’t be an issue but the challenges are:

  1. When building a cabinet out of these materials, we have to pre-confirm the thickness so we can make ensure that the final cabinet width is correct.
  2. If an order is mixing plywood core and composite core EP’s thicknesses may vary too much for some applications.
  3. If an order is using plywood core EP’s and then has 1 longer than 8’ (plywood max length is 8’) and it must be composite core, thicknesses may vary too much for some applications.
  4. A composite core EP beside a plywood core EP may be noticeable if the thicknesses are different.
  5. If an order has plywood core material that varies too much for one sheet to another, this may be too much for some applications.
