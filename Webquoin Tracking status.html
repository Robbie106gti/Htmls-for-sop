The colors mean:
E

  • Yellow = All items have been requisitioned.

  • Green = Ready for optimizing.

O

  • Yellow = The optimizing has started.

  • Green = The optimizing has been completed.

B

  • Yellow = They have started preloading material.

  • Blue= The parts for PA, D and F are complete. Melamine parts aren’t.

  • Green = The entire order is complete.

M

  • Orange when in the Engineering stage, = The order has Millwork (done at the preorder stage).

  • Clear in the Production stage = The order has Millwork.

  • Yellow = The order has been ACAM’d and is ready for the shop to begin.

  • Blue = B has completed it.

  • Purple = That M has completed it and it is ready for Finishing.

  • Blue = That F has completed it.

  • Green = That Millwork has completed it.

PA

  • Yellow = The order has been started.

  • Purple = That PA has completed it and it is ready for Finishing.

  • Green = That F has completed it.

DA

  • Yellow = The order is finished by B and is ready for DA.

  • Blue = The order has arrived from Elias.

  • Purple = Drawer Boxes are assembled and they are ready for Finishing.

  • Green = DA and F have completed it.

D

  • Yellow = The doors have arrived.

  • Purple = The doors have been finished by the door team and they are ready for Finishing.

  • Green = F has completed the doors.

F

  • Yellow = Finishing has started an order.

  • Green = It is completely finished.

FA

  • Blue = The cabinets are assembled.

  • Green = The cabinets have been wrapped (Not necessarily palletized. They may be waiting to be palletized with PA parts).

P

  • Yellow = The Packaging team has started this order.

  • Green = The order is wrapped (and palletized if required).

S

  • Blue = The order is ready but the dealer has decided to not pick it up.

  • Green = The order is no longer in our shop.
