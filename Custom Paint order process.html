Custom Paint order process for the Finishing Team

Custom Paint order process when we “match to an item or sample” chunk of something.

The Finishing team will receive a custom color “match to an item or sample chunk of something” request sheet from the order desk.

  1. On a new PO, note the:
    1. Current date.
    2. Required date (typically 3 working days).
    3. We will P/U (pick up).
    4. Our custom color number.
    5. Order 1 liter of paint.
    6. Note to match to sample.
    7. Note sheen (40 degree (Satin) or 20 degree (Matte))
  2. Give the “item or sample chunk of something” to our purchasing team along with the yellow copy of the PO and three 6” x 10” blocks of the material we want painted, to be dropped off at Richelieu.
  3. Put the white copy of the PO into the purchasing office rack.
  4. Put the request sheet in the custom color request rack in the Finishing office.
  5. Pick up the custom paint when ready.
  6. When we receive the custom paint make 4 sample blocks to confirm the match.
    1. Throw the request sheet away.
    2. In the custom color program, note the Paint code and any instructions.
    3. Get 3 custom color labels from Megan for the Dealers sample blocks.
    4. Make 3 sample blocks for the dealer along with the original “match to item” and give to shipping.
    5. Make 1 sample block for Nickels and put it into the rack in the Finishing office with codes and instructions written on the back.

Custom Paint order process when we “match to an existing paint color eg. Benjamin Moore

The Finishing team will receive a custom color request sheet from the order desk.

  1. On a new PO, note the:
    1. Current date.
    2. Required date (typically 3 working days).
    3. We will P/U (pick up).
    4. Note the color we are matching to (code and name).
    5. Our custom color number.
    6. Order 1 liter of paint.
    7. Note sheen (40 degree (Satin) or 20 degree (Matte))
  2. Fax the PO to Richelieu.
  3. Put the white copy of the PO into the purchasing office rack.
  4. Put the request sheet in the custom color request rack in the Finishing office.
  5. Pick up the custom paint when ready.
  6. When we receive the custom paint make 4 sample blocks to confirm the match.
    1. Match the match to the request sheet and if all is well, throw the request sheet away.
    2. In the custom color program, note the Paint code and any instructions.
    3. Get 3 custom color labels from Megan for the Dealers sample blocks.
    4. Make 3 sample blocks for the dealer and give to shipping.
    5. Make 1 sample block for Nickels and put it into the rack in the Finishing office with codes and instructions written on the back.
