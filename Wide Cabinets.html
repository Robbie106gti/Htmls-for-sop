Current standards for tandem and tandembox:

Also see the Load bearing SOP.

  • Cabinets wider than 39” use the heavier duty (50kg) hardware for the slides.
  • Cabinets wider than 39”use the stabilising hardware.
  • Add screws to the slides on cabinets wider than 33”. 4 at front and 4 at the back.
  • Add screws to the very back for slides longer than 550 (a total of 6 is wanted).
  1. Tandem slides need to be ordered per job.
  2. Stabilising hardware for tandem needs to be ordered per job.
  3. Tandembox needs to be ordered per job. A hardware report would be the best way to go.
  4. Tandembox front stabiliser needs to be ordered per job. A hardware report would be the best way to go.
  5. Stabilising hardware for tandembox should be ordered per job. A hardware report would be the best way to go.
