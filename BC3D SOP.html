Spec’s :

Note.

  • If an item directly beside this cabinet is > then 7/8” proud of the cabinet, a filler is required to compensate. Eg a column that was 1” proud needed a ¼” filler. See Nickels showroom.

Backing

  • Cut mouse hole for access to attaching counter draw bolts etc.
  • Same height as gable. BC3D3930 = 548W x 762H, BC3D3630 = 439.5W x 762H
  • Program = BCD-Back

Bottom

  • BC3D3930 = 976.5W x 976.5L, BC3D3630 = 898W x 898L
  • Program = BCD-B

Gable

  • BC3D3930 and the BC3D3630 = 610W x 762L
  • Program = BCD-EG
  • No Rabbet is needed.

Inner Gable

  • BC3D3930 and the BC3D3630 = 762W x 730L
  • Program = BCD-IG

Top

  • BC3D3930 = 976.5W x 976.5L, BC3D3630 = 898W x 898L
  • Program = BCD-T

Drawer Fronts

  • No 5 piece drawer fronts allowed due to placement of the screws that attach the front to the drawer box.
  • The BC3D drawer front width is dependant on edge detail and material thickness. Drawer fronts are to be drilled on the CNC router. The width sizes are:
  • BC3D39/SQ edge/19mm thick = 357 wide.
  • BC3D39/SQ edge/20mm thick = 356 wide.
  • BC3D39/RD edge/19mm thick = 359 wide.
  • BC3D39/RD edge/20mm thick = 358 wide.
  • BC3D39/HA, CA, CV edge/19mm thick = 360 wide.
  • BC3D39/ HA, CA, CV edge/20mm thick = 359 wide.
  • BC3D36/SQ edge/19mm thick = 337 wide.
  • BC3D36/SQ edge/20mm thick = 336 wide.
  • BC3D36/RD edge/19mm thick = 339 wide.
  • BC3D36/RD edge/20mm thick = 338 wide.
  • BC3D36/HA, CA, CV edge/19mm thick = 340 wide.
  • BC3D36/ HA, CA, CV edge/20mm thick = 339 wide.
