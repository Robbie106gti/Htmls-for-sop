Goals:

Keep the router working constantly.
Supply customers with what they need and how they need it.
The material should remain in good condition.

Machine Checklist:
  • check_box_outline_blank Every morning: Run the Drill test program (scan the label. Minimum material size = 200 x 200mm) to test the drilling depths.
  • check_box_outline_blankEvery Friday: Clean the machine at the end of the day.
  • check_box_outline_blankWhen required: Clean the machine bed.
Job Priority:

Take the next order according to the list of orders on the wall.

Procedure:
  • The piece must be against the machine pins.
  • If a custom program is needed, leave it for Stan to program and machine.
  • If something seems wrong with a program, check plan and see what it say’s. If the piece needs to be re-cut, notify appropriate people. If custom programming is required, put the piece aside and start another. Notify Stan when appropriate.
  • Quality check part. Chipping means that a dull bit needs to be replaced.
  • Rack part according to the following Customer needs.
Customers needs:
When machining Finished Parts:

Separate by:

  1. Job number
  2. Edge tape color
  3. Wall gables-keep the taped edge down.
  4. Material thickness
  5. Always leave at least a 2” minimum space in the rack.
When machining Unfinished Parts:

Separate by:

  1. Job number
  2. Room number.
  3. Wall gables-keep the taped edge down.
  4. Material thickness
  5. Try to keep like pieces together as much as possible. Eg. All stretchers on the end. Etc.
  6. Always leave at least a 2” minimum space in the rack.
Spoil board:

When it gets to a thickness of 14mm, notify Jon so that he can send a new spoil board to Finishing. When you get to a thickness of 10.5mm, replace the spoil board with the new one.

Bits:
  • When bits are changed, run the appropriate bit test program.
  • The operator needs to let Jon know when the drawer has only:
    • Two Half Composite (HC / 3189) bits remaining.
    • One Plywood (PLY / 3489) bit remaining.
    • One 3/8 (1340) bit remaining.
  • Every Thursday morning, check all bits and see if any need to be sharpened. Give any dull bits to Jon before lunchtime.
