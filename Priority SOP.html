Priority Boards

  1. The basic setup for all shop Priority boards are set up with the same format.

  2. Listed by Production End date.

  3. Work from the top down unless instructed.

  4. Notes on the right with any issues.

  5. Cross off when done.

  6. If the order is complete but there is an issue (done but see notes), cross off the job # and put notes on the right side. Only cross off the notes when the offending issue has been resolved.

  7. If the priority must change, put numbers on the list (Keep them in order of production end date) indicating the offending order.
