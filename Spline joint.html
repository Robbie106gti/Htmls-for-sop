We want to eliminate nailing strips on header returns, gable fillers etc.:
  1. so installers don’t need to worry about cutting into staples or
  2. Stop the nailing strip from being in the way.:

Use the Spline joint for Header return scenarios and use biscuits for larger situations… eg. a tall pantry gable filler.

Using the Spline joint

  • For mouldings with returns. For wider items, we'll continue to use biscuits.

  • Use a full length spline, 22mm W. x 8' Long x 3mm thick (We often receive cover sheets that can be used for this).

  • 15 min. / piece and dry time

  1. Put Spline blade into saw
  2. Adjust height using Spline height gauge. Width = 3mm.
  3. Mark parts so you know faces you're working on
  4. Break edges (microbevel)
  5. Cut appropriate parts.
  6. Put glue in grooves
  7. Insert spline and join
  8. Clamp and let dry.
  9. Unclamp and stack
