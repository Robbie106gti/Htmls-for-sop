Purchasing items with an order tag.

Put the appropriate Tag in the “In” Order box, in the Purchasing office. All standard stock items should make it into the shop in 3 working days, except cardboard, 1” plastic spacers and Blum Canada take 5 days. If it is a rush and is expected quicker then this, give the tag to the Purchaser directly and tell him when you need it. Submitting tags to Purchasing should only be done by Lead hands or Team Leaders. The Purchaser will put the tag in the “Out” box after ordering. These can be picked up after the item arrives.

Purchasing items without an order tag.

Use one of the generic Pink slips in the Purchasing office and leave it in the “In” Order box (give to Purchaser directly if it is a rush). Whenever possible, give Purchasing the order code. The Purchaser will put the tag in the “Out” box after ordering. These can be picked up after the item arrives.

Stock order tags

Colors:
  • B=Pink
  • D=Pink
  • DA=Green
  • DP=Yellow
  • F=Pink
  • FA=Yellow
  • M=Orange
  • PA=Light blue
  • S=Yellow

Goals :

  • To keep less then 1 month of any stock material in shop. Err on the side of less, then more. Don't order more then a single month supply of items for stock. Any veering away from this rule must be OK'd by Ted.
  • To have an easier method of seeing all we have in stock.
  • To have a visible reminder on as much stock material as possible, as to when to reorder, so everybody knows.
  • To make reordering of stock material as automatic as possible.
  • Get Purchasing a price for everything. One of items from Home depot etc. don’t need to go into Purchasing Plus but the receipt must go to Accounting.
  • Clean up old stock. Nothing should be on the shop floor that isn’t standard inventory or a custom order labeled for an upcoming order. All else should be either:
    1. given to Purchasing to return
    2. put into the Craigs list area
  • All items to be purchased by Purchasing. The only exceptions are custom finishes and “one of” items by Maintenance. See the F and M team information (further down the page) for specific details on these items.

Tag layout:

Note that if there is a hole punched in the corner of a tag, this notes that the item is a 2 day minimum pick up time. This is consistent for a lot of Richelieu finishing products.

Procedure:

  • The stock order tags will be in a rack in Purchasing, if they are not in an obvious location by the actual item you want to reorder.
  • When an item arrives, it should be checked ASAP for any issues. Purchasing will check for correct part numbers and package damage.
  • Items that are for specific people should be immediately labeled with their name. eg gloves.
  • Any items in your Teams area that you aren’t using should be returned to the supplier. If that isn’t possible, put them in the Craig’s list zone.
  • Any items in your Teams area that you aren’t using but you think you might, should be Blue Tagged and put in the Craig’s list zone so it can be reviewed in a year.

Items that become unusable:

If for whatever reason, a custom order, or a stock item becomes unusable, it should be brought to Jason’s office or, if it’s to big, be brought to his attention with an email so he can send it back. If for some reason it can’t be sent back, Jason will track it to use it in the future or we will sell it ASAP on Craigs list.

Reasons for all this:

  • Keep our “cash sitting around” to as small an amount as possible.
  • Some items get to old to use. Out of style or unusable.
  • Keep required storage space down.
  • Use the first in, first out mode so we can track supplier/item quality and style. CONSISTENCY.
  • If we decide to stop using something, we don’t want a large supply remaining (yellow gloves).
  • Bifold door hardware. We actually had it in stock, but dispersed throughout the shop.
  • Book keeping needs to know so we can properly balance the books and actually know we aren’t getting overcharged.
