Goal:

Getting the Aventos power factor correct every time.

Facts:

  • The power thing required is only an educated guess and turns out to be wrong 15 ish% of the time.

  • To get the fix/reorder of the correct power factor to be painless, we will order in an "escape inventory" of 3 sets of each for issues and replace as required. (Note that we will do this for the HKXS and HF. The rest we will only do if we notice repeated issues).

Rules:

  1. We will continue ordering the Aventos Lift Mechanism per job and only use the "escape inventory" when the order doesn't work.

  2. When the "escape inventory" builds up to more than 3, we can use the excess for other jobs.
