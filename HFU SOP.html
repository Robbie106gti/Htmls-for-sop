HFU process:

OE

  • All custom HFU’s get a complete AutoCAD drawing signed for confirmation by the dealer.

  • Note if “cutout needs to be exact”. Exact means PA will make a template that matches the liner and make sure it just fits into the opening.

  • Note if “cutout needs to be exact” (these are typically Ventahoods), the opening will be 1mm longer and wider than the actual liner.

E order processing

  • Photocopy & highlight HFU’s and give them a production day 2 days before the order (Put the photocopy in its own red folder and in the “Ready for optimizing” drawer 2 days ahead of the job).

E optimizing

  • Note material thickness of: MDF = 16mm, veneers = 15.6mm.

  • All HFU’s get optimized individually with a production date of 2 days before the rest of the order.

  • Allow for edgetape.

B

  • Panel Saw-cut all parts/label/machine/dowel/edge tape.

PA

  • Assemble

  • Note if cutout needs to be exact, make a template that matches the liner and make sure it just fits into the opening.

  • Tell Harry how long it takes.

F

  • Finish

S

  • Ship

Fire retardant laminate:

  • The option to have the fan area of the HFU laminated with a fire retardant laminate. Suppliers = WA and Formica. Colors = Black, White and Maple. The very bottom edges of the HFU need not be laminated with this material. The cost for this is $250.

Removable panels:

Removable panels that are vertical (HFU6):

  1. Wainscot panels will use a block system as per drawing. The block length will be half the length of the removable panel and it will be centered.

  2. Not wainscot panels will use a ball catch system in all corners.

  • All removable panels must be a snug fit.

  • This is only for a removable panel that is not on an angled front. The angled ones don’t require the ball catches or block set up for the wainscot panels.

  • All removable panels must be taped into position for shipping.
