Euro Cargo and its spacer

Waste bin, Garbage can, Recycle containers/hardware.

Situation:

Note that a lot of the Euro Cargo hardware requires a cabinet width that follows the metric system e.g. 350mm, 400mm, 450mm, 500mm, 550mm, 600mm etc.

Our codes RGB-POD18, RGB-POD2B24, RGB-POD4B24 need a spacer.

The solution to this is:

  1. Use a spacer to allow the hardware to work following our imperial width cabinets.
  2. Reduce the cabinet width to the recommended metric width.

We will buy spacers from Alliance plastics.

Spacer-4.5 = SWS2270A Approx. 78.63/ 1000

(Back up solution... make awesome in house metal spacers (RGB-POD spacers 3mm and RGB-POD spacers 4.5mm) by Bob.)

When a spacer is required, use the plastic dowels in the predrilled holes and use ¾” screws to fasten the hardware and spacers to the gable.
FA has been given tags so that they can automatically order more when required.

RGB-POD Spacer:

Euro Cargo (RGB-POD12, RGB-POD18, RGB-POD2B24, RGB-POD4B24) Installation steps :
  • Attach sliding bracket to slides.
  • Attach bumpers just above lid supports.
  • Attach frame to top of sliding bracket. Manually center and ensure that the frame is parallel with the front of the cabinet.
  • Attach DFS to hardware panel.
  • Attach door to hardware panel.
  • Attach brackets to hardware panel. Use long system screws in the predrilled holes.
  • Attach hardware panel to sliding bracket using the 4 machine bolts, 2 per side, but don’t completely tighten them yet.
  • Screw in and tighten all wood screws to attach the bracket and the frame to the hardware panel.
  • Final adjust the door tilt and lean with the adjustment screws on the sliding brackets.
  • Tighten the 4 machine bolts, 2 per side, on the sliding brackets.
  • Attach cover caps.
  • Put lid in place.
  • Insert Garbage cans.
  • Put manufacturers installation instructions into the garbage can in case the installer has questions.
