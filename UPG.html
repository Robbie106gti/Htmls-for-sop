Standards:

  • Thickness - ¾”
  • Sheets = 9’L x 4’W. We would get this cut down to 9’L x 42”W for us.
  • Minimum width = 2” with edgetape.
  • HG highest = 48”. Higher than this will be attached with dowels. Bob will test.
  • Stiles- tape the front edge with edgetape, side by side if necessary.
  • We will be ordering all parts from them cut to size and edgetaped (laser) when ever possible.
  • We will have to order edgetape for the cabinets.
  • Toekicks, we will play by ear. If we need a lot, order sheets and if very few, order individual parts.
