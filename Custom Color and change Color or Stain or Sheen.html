PROMO-S (Promotional Block Stain) and PROMO-P (Promotional Block Paint)

Custom Paint Color procedure:

  • Whenever a customer wants to match any color (i. e. from a fan deck, from a site sample, etc…), order desk needs to make sure we don’t have the color in our catalogue before creating a new custom number. If it is an existing color, painted more than one year ago, there is no need to order a color match, but just a freshly painted set of sample blocks;

  • Any color discontinuation, deletion or cancellation means that the color cannot be ordered for future jobs, even if the job is an extension of an old job finished with a color within the old number system.

  • Even though we will be discontinuing colors we should keep files for all of them with the color number, the code / recipe from the supplier and the color description (job number, customer / dealer’s name, date off order and whatever we consider relevant)

Custom Stain procedure:

  1. Order Desk receives a site sample from a dealer.

  2. ASAP, evaluate for standard or premium pricing before giving the custom number:

  3. Give the stain an appropriate number in either the S or W custom numbering program.

  4. ASAP, Order Desk can send the order to scheduling. At the same time, Order Desk will email Finishing (Shyamal) so he can start the color match process

    • It is Premium pricing when:

      • The custom stain is being applied on a non standard wood.

      • The custom stain is being applied on maple.

    • All other scenarios can be considered standard pricing.

  5. Finishing will send the sample to Richelieu with some test blocks ASAP.

  6. Richelieu’s result is QC’d and applied to sample blocks.

  7. Blocks are labelled and shipped to the Dealer.

Notes:

  • Order Desk will submit these color matches well in advance of the actual order, whenever possible.

  • We will, whenever possible, make all stains using our system: tone, spray and wipe.

  • Timeframe =

    1 day at order desk
    5 days at Richelieu.
    2 days in transit
    2 days shop processing

Custom Paint or Stain Color #’s:

When the Custom Color was last used > year ago make new Promo Blocks for the customer and the shop. No setup fee required. Use the same custom color number.
When the Custom Color is the old numbering system, make new Promo Blocks for the customer and the shop. No setup fee required. Create a new custom color number and in the notes refer to the old number. In the notes of the old color number, refer to the new number.

Custom color samples:

Custom color samples must be approved before an order is sent to scheduling, unless otherwise instructed by the production manager.

When a Finishing change (color, stain or sheen) is made to an existing sample:

When a change is made to any existing sample, the color will be affected. Therefore we must:

  • Notify the customer of the change.

  • Make a new sample and ship it to the customer so they can confirm it.

  • Make a new sample and ship it to the customer even if they say they don’t need one.

  • Make a new sample for our finishing quality control check.

Custom Stain/Paint Setup fee: SOP

  • With this request, three samples will be supplied. Two are for the dealer (He can give one of these to the customer.) and one is for our Finishing Team. These stain samples will be 4”W x 6”L solid block. These paint samples will be 4”W x 6”L x 3/16” thick material.

  • When an old stain is used from our Stain program, a setup fee isn’t required when the stain formulae begins with the letters AR, meaning that we get it from the Richelieu program.

Matching a color that Nickels Cabinets has discontinued:

  • If it is a stain that is less than 1 year old, no charge. We will provide sample blocks for approval.

  • If it is a stain that is greater than 1 year old, there is a charge. We will provide sample blocks for approval.
