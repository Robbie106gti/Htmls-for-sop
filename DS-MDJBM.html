Standards:

16mm thick melamine for the bottom’s, ends and sides.

The melamine matches the cabinet interior.

Blind dado joints.

No labels on the parts are required. Just need to know which job(s) are in the batch.

Process:
B:
  1. Run test to ensure that the machine is setup properly.
  2. Write the appropriate job #’s on the test pieces and send them with the parts.
  3. QC material. Cut parts on P2P nesting machine.
  4. Sort into bottoms, sides and ends
  5. Sort by size also if time allows.
  6. Tape all appropriate edges. Use the test pieces to check that the edgebander is running properly.
DA:
  • Assemble as per MDJBM process flowchart.
