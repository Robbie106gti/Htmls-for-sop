Single soap tray with hinges. Two soap trays with hinges.

The MSC-SST-WH will not work in a cabinet with a header at the top of the cabinet. The header would have to be removed.

Cabinet Minimums

  • MSC-SST-WH with an 11” soap tray

    • Minimum opening required is 11 3/4”

    • Minimum cabinet width (not a stile or DD application) required = 13”.

    • Minimum cabinet width required, with a stile or DD application = 27”

  • MSC-SST-WH with a 14” soap tray

    • Minimum opening required is 14 3/4”.

    • Minimum cabinet width (not a stile or DD application) required = 16”.

    • Minimum cabinet width required, with a stile or DD application = 33”.

  • SC-2ST-WH with an 11” soap tray

    • Minimum opening required is 22 3/4”.

    • Minimum cabinet width (not a stile) required = 24”

    • Minimum cabinet width required, with a stile = 27”

  • MSC-2ST-WH with a 14” soap tray

    • Minimum opening required is 28 3/4”

    • Minimum cabinet width (not a stile or DD application) required = 30”

    • Minimum cabinet width required, with a stile = 33”

Hardware
Drawer front Flap Stay:

  • SST3

  • Attach on the left side of the cabinet when possible. When it’s a DD cabinet and both drawer fronts have soap trays, attach on the left and right gable.

  • Both gables will have the pilot holes for the SST3 hardware.

  • Only 1 per drawer front.

  • From Richelieu. Product code = 57512130

Drawer front hinge

  • HINGE-F

  • Product code = 71M055180. Richelieu.

  • 2 per drawer front.

Hinge Mounting Plate

  • HINGE-MP

  • Product code = 173L8100 Blum.

  • 2 per drawer front.

Team responsibilities to make the MSC-SST-WH and the MSC-2ST-WH happen:
B

P2P machine the stretcher for the mounting plates.

D

Drill the drawer front for the hinges and the pilot holes for the hinge screws and the SST3 hardware.

FA

When there is a MSC-SST-WH:

  1. Attach the HINGE-MP mounting plates to the stretcher using system screws
  2. Attach the SST3 flap stay hardware to the gable where indicated by the pilot holes using 5/8” wood screws. Remove the locking washers and only use the Teflon washers.
DP

Attach the HINGE-F to the drawer front with screws.

Clip the drawer front to the mounting plates and attach the SST3 to the drawer front using the pilot holes.

Attach the soap tray to the drawer front.
