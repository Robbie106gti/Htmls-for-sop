Batch Creation:

  1. Print out Tracking Status Report.

    • Batch no more than 3 day’s work or $60,000 worth of jobs. Do not exceed.

  2. Pull jobs from file cabinet.

  3. Go to S:Quickcam CSVout and create folder using batch name.

    • Batch name is the earliest production date on the tracking status report.
      Example…(12A)

QuickCAM:

  1. Export jobs from island computer station

    • S:Quickcam CSVout (shared S drive with Stan’s computer)

  2. Import jobs using QuickCAM on Stan’s computer station.

    • Run all of the jobs, one at a time. This is to test edge tape.

  3. Use Reports program to check if edge tape is OK.

    • If an edge tape shows as “Custom”, then add edge tape to list.

    • Go here to change: QuickCAM-OpenDataStore-NickelsData-ColorCombo_Edging

    • Re run job in QuickCAM and check Reports to see if edge tape shows up.

  4. Go to S:\Quickcam CSVout\ and cut and paste exported jobs into appropriate batch folder.

WordPad:

  1. Go to S:\Quickcam CSVout\(batch #) and open any job that has a custom cabinet that needs part modification.

  2. open the custom cabinet job using WordPad.

  3. Delete any extra parts.

  4. Change any sizes that need to be modified.

  5. Save

CADcode:

  1. Open CADcode and press the Mapped Data Reader button.

  2. Load the jobs for the batch, name it the batch #, run program.

  3. Press the Control Panel button.

  4. Leave checked the materials that are labeled Saw to CNC . Uncheck all other items.

  5. Find gables and items that need to be customized.

    • Find these items material type and lettering from QuickCAM.

  6. Open custom item in CADcode and an image appears.

    • Right click and choose edit in AutoCAD.

  7. Edit in AutoCAD and in CADcode right click on image and choose Update from AutoCAD.

  8. When all editing is complete choose Run Job in CADcode.

Paperwork:

  1. Go to Reports program:

    • Print Edge Tape

    • Print Color Labels (Solid Parts (labels), Office Copy (finished parts)).

    • Print Batch Reports

  2. Go to CADcode program:

    • Print materials in order as on Material Summary Report.

    • Press the View In Panel Optimizer button to call up the printing screen for each material.

    • Choose multiple patterns per page when there are 3 pages or more to printed. Then print the last pattern on one page.

  3. Paperwork order:

    • Material Summary Report , Material Patterns, Tracking Status Report all stapled together.

    • Breakout shop paperwork used in optimizing.

    • Finished Parts paperwork.

    • Tracking Status Report

    • Solid Parts Labels

    • Edge Tape List

    • Attach all this paperwork together with large clip.

    • Bring to panel saw.

End:

  1. On Stan’s computer desktop run Move Files. Bat.(moves panel saw, Rover A, Rover B, Label Pictures and Label Info data to the right folders)

  2. On Stan’s computer desktop run Fix Program.
