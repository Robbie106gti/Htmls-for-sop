DS-WT GT ST---DRI-FILE (Tandembox with a file drawer)

  • DRI-FILE-D ---Maximum cabinet width of 24” for the standard setup in white metal drawer boxes. If the cabinet is wider, we can center our standard setup leaving equal spaces on either side of the file system. If you want to maximize the useable space, we can put in 2 file drawer systems, halving them in the drawer box. Putting in two file drawer systems must be specifically requested.

  • If the screws of the file drawer are positioned to go through the front of the drawer front, a piece of melamine 2⅜” wide x the same length as the drawer end will have to be attached to the back of the drawer front so the file system can be attached to it.

  • DRI-FILE-W ---Maximum cabinet depth of 27” for the standard setup in white metal drawer boxes. If the cabinet is deeper, we will offset our standard setup keeping the files at the front. If you want to maximize the useable space, we can put in 2 file drawer systems, halving them in the drawer box. Putting in two file drawer systems must be specifically requested.
