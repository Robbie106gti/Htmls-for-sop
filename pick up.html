Pick up SOP
  • Purchasing will do all standard pick ups when possible.
  • Shipping, Service and Finishing will tell Purchasing when they are on the road so they can do pick ups when appropriate.
  • Shipping, Finishing, Doors and Maintenance will notify Purchasing if they need something picked up or dropped off.
  • Finishing will notify Purchasing when Finishing has to do a special pick up so they can pick up or drop off others if appropriate.
  • All teams will notify Purchasing if they have an errand that needs to be done, to see if it fits into his pick up schedule.
  • Acroma will call Purchasing when they have a pick up. Finishing will put a note on their PO to Acroma to tell them to do this.
  • If a pick up is a rush, Purchasing needs to know the exact time it is required by.
Pick up from a supplier:
  • If you’re picking up at Richelieu, you need an order number. Call Richelieu 1 hour ahead so they can have it ready.
  • Make sure that the packing list matches what you’re picking up and that it has been fulfilled.
  • If an item is not there, ready or incorrect, please call purchasing before leaving the supplier to see if the problem can be rectified before you leave.
  • Leave the picked up items in one of the specific receiving areas.
  • Give Purchasing the packing list.
  • If you don’t receive a packing list, Purchasing must be told.
  • If there is only a receipt for an item only ordered once, give this receipt to Accounting. Purchasing doesn’t need to get involved with these items.
